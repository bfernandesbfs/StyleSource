# StyleSource

The Swift code generator for your theme.

## Installing

Make sure Xcode 10.0+ is installed first.

### Make`

```shell
git clone https://github.com/bfernandesbfs/StyleSource.git
cd StyleSource
make
```

### Swift Package Manager

**Use as CLI**

```shell
git clone https://github.com/bfernandesbfs/StyleSource.git
cd StyleSource
swift run StyleSource
```

## Support

 * Colors
 * Fonts
 * UI
